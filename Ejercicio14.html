Almacenando información en una base de datos

Descripción

Esta aplicación consiste en que al seleccionar un archivo de texto desde nuestra máquina cliente, el contenido del mismo se almacenará en una base de datos y se mostrará la información que alberga la misma en un recuadro, usando la API File y la Api IndexerDb. Ademas disponemos de un botón que nos permite poner la página web en pantalla completa.
